# Slp explorer

#### Easy way to track slp transaction and token in bitcoin cash

## Backend
SLP explorer work with backend api build by nestjs 
(https://github.com/salemkode/slp-explorer-backend)[sorce code]

## Explorer Pages

#### Token Page

```
/token/:tokenid
```

#### Transaction Page

```
/tx/:txid
```

#### Address Page

```
/token/:address
```
